# gmscript-clear-chinesesearch
a greasemonkey script to clear  chinese search result page. 

# 介绍

这是greasemonkey内运行的一个脚本,作用是清理某中文搜索引擎结果页上的广告. 
因为它们家的广告确实太恶心了. 也可能是孤的心理有强迫症,哈哈!


# 如何实现
就是用js找到页上的一些节点, 之后清理节点内容.
没有使用jquery,因为懒的去引入.
